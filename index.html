Research Hub

Spacekime project materials in one place.

This site gathers the current Spacekime videos, audio, diagrams, and papers into a single GitHub Pages experience so a visitor can move from the big-picture architecture to the supporting documents without leaving the browser.

Across the collection, the project appears to explore Spacekime as a conceptual framework with extensions into structure, incompleteness, collapse, consciousness, and theology. The homepage is organized to help a first-time reader start visually, then move into talks, audio, and the longer papers.
